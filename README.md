# docker-and-kubernetes

### What is Docker
Docker is a tool for running applications in isolated environment.

### Docker Container vs VM
Container Benefits: 
1. Run container in seconds.
2. Less disk space.
3. Use less memory
4. Do not need full OS.
5. Deployment
